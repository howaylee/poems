# 📖 小学生必背古诗75首 — 智能背诵助手

> 一个纯前端的古诗背诵学习工具，帮助小学生高效完成 75 首必背古诗的背诵任务。

🌐 **在线体验**：直接用浏览器打开 `index.html` 即可使用

---

## 功能总览

### 一、📋 古诗列表（主页）

| 功能 | 说明 |
|------|------|
| **75首完整古诗** | 涵盖汉、南北朝、唐、宋、元、明、清各朝代经典 |
| **卡片式展示** | 每首古诗以卡片形式呈现，显示编号、标题、作者、前3句预览 |
| **状态标记** | 已背诵（绿色左边框）、复习中（橙色左边框）、未学（默认） |
| **快捷操作** | 每张卡片上直接「标记背诵」/「标记复习」，无需进入详情 |

### 二、🔍 搜索 & 筛选

| 功能 | 说明 |
|------|------|
| **全文搜索** | 输入关键词，同时搜索**标题、作者、诗句内容**，实时过滤 |
| **状态筛选** | 4个标签页一键切换：全部 / 未学 / 复习中 / 已背诵 |

### 三、🙈 遮罩背诵模式（核心亮点）

点击任意古诗卡片 → 进入详情弹窗 → 点击「🙈 遮罩背诵」

- 每个汉字变成一个**紫色色块 + 小圆点占位**，完全看不到字迹
- **点击单个字** → 逐字揭示答案（想不起来才看）
- **点击整行** → 显示该行完整诗句
- 背完后可切回「📖 正常模式」对照原文
- 配合「⏭️ 下一首」快速切换

> 💡 使用场景：自己先背，想不起来的字点一下看提示，真实检测背诵效果

### 四、🎲 随机复习

- 点击顶部「🎲 随机复习」按钮
- 从所有已标记（背诵/复习）的古诗中**随机抽取一首**
- 自动打开弹窗，建议配合**遮罩模式**使用
- 每次点击都是惊喜，避免"总是背同一首"

### 五、📅 15周背诵计划

| **完成高亮** | 全部背诵完的周变为绿色背景 ✅ |
| **点击跳转** | 点击某周自动打开该周第一首未背诵古诗的详情 |

### 七、📊 统计面板

| 功能 | 说明 |
|------|------|
| **三色统计卡** | 已背诵 / 复习中 / 未开始，大数字展示 |
| **进度条** | 三种状态的百分比可视化 |
| **智能建议** | 根据当前进度动态给出学习建议 |

#### 智能建议规则

| 背诵数量 | 建议 |
|---------|------|
| 0 首 | 还没有开始背诵哦！建议从第1周计划开始，每天背诵1-2首古诗。坚持就是胜利！💪 |
| <15首 | 已经背诵了N首啦！继续保持每天学习的习惯，按照15周计划稳步推进吧！ |
| <45首 | 已完成N首！可以开始定期复习已学内容，利用「随机复习」功能巩固记忆。 |
| <75首 | 太棒了！已完成大部分(N/75)，坚持完成最后部分，加强复习巩固！🎉 |
| 75首 | 🎊 恭喜完成全部75首！请定期使用随机复习功能巩固记忆，温故而知新！ |

### 八、⌨️ 键盘快捷键

在弹窗打开时：

| 快捷键 | 功能 |
|:------:|------|
| `←` | 上一首 |
| `→` | 下一首 |
| `Space` | 标记/取消背诵 |
| `R` | 标记/取消复习 |
| `Esc` | 关闭弹窗 |

全局快捷键：

| 快捷键 | 功能 |
|:------:|------|
| `/` | 聚焦搜索框 |

### 九、🔧 数据管理

| 功能 | 说明 |
|------|------|
| **本地存储** | 所有进度保存在浏览器 localStorage，刷新不丢失 |
| **导出进度** | 一键导出 `.txt` 格式报告，包含已背诵/复习中/未开始清单 |
| **重置进度** | 一键清空所有记录（有确认弹窗防误操作） |
| **版本升级** | 存储格式 v2，兼容旧数据迁移 |

### 十、🎨 UI 设计亮点

- 渐变色 Header（紫→粉），中国风韵味
- 卡片 hover 上浮动画 + 阴影加深
- 进度条流光动画效果
- 弹窗滑入 + 背景模糊（backdrop-filter）
- Toast 操作提示反馈
- 完全响应式，手机/平板/电脑均可使用
- 底部固定导航栏：列表 / 计划 / 朝代 / 统计

---

## 技术实现

- **纯前端**：单个 HTML 文件，无需后端服务器
- **数据持久化**：localStorage 本地存储
- **响应式设计**：适配手机、平板、桌面端
- **无依赖**：原生 HTML/CSS/JavaScript，开箱即用
